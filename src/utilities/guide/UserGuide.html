Creating a Recipe or Meal

In the menu bar, locate the Edit menu item.
Then click on the Edit button.
From the Edit option, two options should appear: Recipe and Meal.

Choose the desired option.

The recipe and meal editors have 5 button options:

  1. The New Button
  2. The Open Button
  3. The Save Button
  4. The Save As Button
  5. The Close Button

In the Recipe Editor

Next to the button options enter the Name of the Recipe and the serving size in number of people.

Utensils

  1. Type the name of the utensil needed.
  2. (Optional) Add any details about the needed utensil. Ex. Wooden for Wooden Spoon
  3. Click the add button.

Ingredients

  1. Type the name of the ingredient needed.
  2. (Optional) Add any details about the needed ingredient. Ex. Ripe for Tomato
  3. Type the amount of the ingredient.
  4. Select the units of from the dropdown menu.
  5. Type the number of calories.
  6. Type the density of the ingredient.
  7. Click the add button.

Steps

  1. Select the action being done from the dropdown menu.
  2. Select what is being used either utensil or ingredient.
  3. Select the utensil being used.
  4. (Optional) Add any details about the step. Ex. Hot for Heat
  5. Type the number of minutes the step would take.
  6. Click the add button.

Delete

  1. Highlight or double click on item to be removed.
  2. Hit the delete button.

In the Meal Editor

  1. Type name of the meal.
  2. Click the add recipe button.
  3. Select recipes to use in the meal.

Delete

  1. Highlight or double click on item to be removed.
  2. Hit the delete button.

View Shopping List

Click on view in the menu bar and select Shopping List Viewer

  • Select a recipe or meal file.
  • Enter serving size.
  • View the shopping list and check off if youve purchased whats listed.

View Process

Click on view in the menu bar and select Process Viewer

  • Select a recipe or meal file.
  • Enter the plating time of cooking.

Use the Calorie Calculator

Click on tools in the menu bar and select Calorie Calculator.

The calorie calculator has two buttons:

  • Calculate
  • Reset

  1. Enter the necessary information: Ingredient, Amount, and Units
  2. Click the calculate button.
  3. To reset and clear the calculator, click the reset button.

Using the Units Converter

Click on tools in the menu bar and select Units Converter.

The Units Converter has two buttons:

  • Calculate
  • Reset

  1. Enter the required information: Units being converted from, Units being converted to, the ingredient being used, and the amount being used.
  2. Hit the calculate button and the result will appear in the result space.
  3. To reset and clear the converter, click the reset button.
